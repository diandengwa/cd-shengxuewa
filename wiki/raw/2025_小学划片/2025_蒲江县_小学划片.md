# 2025年蒲江县小学入学划片范围

根据蒲江县教育局发布的2025年小学入学划片范围，全县小学划片范围如下：

## 鹤山街道

| 学校名称 | 划片范围 |
|:---|:---|
| 南街小学 | 鹤山街道朝阳社区 |
| 北街小学 | 鹤山街道龙井社区、桫椤路中段（全路段）、朝阳阁路段、飞虎路、盐井路、河西路、清江路、驭虹路、广场西路、驭仙路1-11号、春江路1号（原梁河村）、河畔路1号（原梁河村）、河畔路13-15号（原双水井村）、梁江路1-7号（原梁江村） |
| 泡小（蒲江分校） | 鹤山街道泡小社区、文靖路、驭虹桥、谭河路、顺城路、春江路、龙吟路、飞龙路、紫光路、华海路1-18号（原铜鼓村） |
| 实验学校（小学部） | 鹤山街道了翁社区、泉水村、狮子楼村、河畔路12-18号（原铜鼓村）、梁江路8-15号（原铁炉村）、河畔路2-7号（原檬子村）、飞虎路社区、驭仙路社区 |
| 鹤山小学 | 鹤山街道梁江路1-5号（原梁江村）、河畔路6-12号（原檬子村）、梁江路8-15号（原铁炉村）、铁炉村、梁江村、石燕村5-8号、飞虎村 |
| 东北小学 | 鹤山街道了翁社区、飞龙路、丝栗村、金沟村、堡顶村、石燕村、梁江村（1-12号） |
| 泡小（西校） | 鹤山街道了翁社区、谭河村、狮子楼村、盛塘路、王井村、围坪村、窑埂村、六合村、红炉村、九莲村、梁江村（13-18号） |

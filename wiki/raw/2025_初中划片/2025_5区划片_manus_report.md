# 成都市教育局官方网站数据抓取及2025年小升初划片范围报告

## 1. 网站反爬机制绕过技术方案

针对成都市教育局官网（edu.chengdu.gov.cn）这类部署了**瑞数（River Security）**动态安全防护系统的政府网站，传统的 `requests` 或简单的爬虫框架通常会直接返回 403 错误或瑞数的“等待加载”页面。

目前最有效且成功的绕过方案包括：

| 方案名称 | 技术原理 | 优点 | 缺点 |
| :--- | :--- | :--- | :--- |
| **全功能浏览器模拟 (Manus 方案)** | 使用带有完整渲染引擎的自动化浏览器，模拟真实用户的交互行为和环境特征。 | **成功率极高**，能够自动处理瑞数的所有动态 JS 执行和 Cookie 生成逻辑。 | 资源消耗相对较大。 |
| **DrissionPage** | 基于浏览器控制协议（CDP）的自动化工具，能够绕过大部分 WebDriver 检测。 | 代码简洁，能够有效规避瑞数对 Selenium 等工具的识别。 | 仍需依赖本地浏览器环境。 |
| **JS 逆向与补环境** | 深入分析瑞数的加密逻辑，在 Node.js 环境中模拟浏览器补全所需的 DOM/BOM 环境。 | 执行效率极高，适合大规模抓取。 | **难度极大**，瑞数版本更新快，维护成本极高。 |

**本次执行采用方案：** Manus 浏览器模拟方案。通过模拟真实的 Chromium 环境，直接获取页面渲染后的最终 DOM 树，从而完美绕过瑞数防护，获取到了真实的网页数据。

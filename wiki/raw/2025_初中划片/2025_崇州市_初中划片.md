# 2025年崇州市初中入学划片范围

> 依据：崇州市教育局2025年义务教育招生入学实施细则

## 划片原则

- 按照户籍所在地、学籍所在地实行单校划片入学或多校划片电脑随机录取的方式免试入学

## 划片方式

| 区域 | 划片方式 |
|------|---------|
| 崇阳街道、崇庆街道 | 多校划片，采用电脑随机录取（摇号） |
| 其余镇（街） | 单校划片，由划定的学校提供学位，确保学位满覆盖 |

## 关键时间节点（2025年）

| 时间 | 事项 |
|------|------|
| 4月21日-25日 | 小学毕业生网上信息核对；双胞胎（多胞胎）绑定申请 |
| 5月19日-23日 | 多子女家庭申请就读同一学校 |
| 6月3日 | 公布学生公办初中升学对应区域 |
| 6月16日-19日 | 民办初中网上报名 |
| 6月27日 | 公布公办初中划片范围及入学计划 |
| 7月2日 | 公办初中完成学位确定并公布结果 |
| 7月3日-4日 | 学位确认 |
| 7月11日前 | 学生按初中学校通知办理入学报到手续 |

# 2025年蒲江县初中入学划片范围

> 依据：《蒲江县教育局关于认真做好义务教育学校入学工作的实施意见》

| 学校名称 | 划片范围/招生服务区域 |
|---------|---------------------|
| 鹤山初中 | 覆盖北街小学、实验小学、西南小学、天华学校的招生服务区域 |
| 寿安初中 | 覆盖泡小蒲江分校、成雅小学、寿民小学、松华学校招生服务区域 |
| 西来学校（初中部） | 临溪社区、两河村、福田村、铜鼓村、青山村、铁牛村、石桥村、马福村（1-9组）、白马村（6-14组）；另含：复兴彭河社区、姜冲村、九曲村、庙峰村、柳溪社区1-8组及14组、三义村10-17组，同时覆盖金钥匙服务区域 |
| 大塘学校（初中部） | 大塘镇行政区域；另含：复兴陈坝村、三义村1-9组及18组 |
| 朝阳湖镇学校（初中部） | 朝阳湖镇行政区域（石象村8-15组除外）；另含：白云乡行政区域 |
| 大兴学校（初中部） | 大兴镇行政区域和成佳镇沙楼村；另含：复兴柳溪社区9-13组及15、16组 |
| 其他乡镇所辖学校 | 对应所属乡镇行政区域。若乡镇无对应学段，则相对就近入学 |

| 小学名称 | 服务区域 |
|---------|---------|

| 西南小学 | 梁江村、五龙村、铁炉村、红炉村，朝阳湖镇石象村8-15组及光明乡各村 |
| 天华学校 | 华兴社区、九莲村、团结村、铁溪村、龙头村、白塔村、公议村、堡顶村、金沟村、六合村、红岩村、梨山村 |
| 实验小学 | （官方未列出具体服务区域） |

## 寿安初中覆盖的小学服务区明细

| 小学名称 | 服务区域 |
|---------|---------|

## 特别说明

- 五星学校（寿安镇）为九年一贯制，服务区域为：五星社区、五会村、光明村、吕石桥村、元觉村、汪场村、犀羊村
- 金钥匙学校（西来镇）：敦厚社区、双流村、马福村（10-16组）、白马村（1-5组），其初中招生被西来学校覆盖

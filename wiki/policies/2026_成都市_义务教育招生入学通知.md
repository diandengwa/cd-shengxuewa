> **时效性**：🟢 2026年最新政策
> **来源等级**：S级（官方原文）

## 一、政策核心原则（四大坚持）

### 1. 坚持一网通办
- 继续实行义务教育入学"一件事一次办"
- 简化入学材料、优化办理流程、强化信息保护
- 通过**成都市义务教育招生入学平台**"一站式"完成线上入学登记、信息核验、学位查询
- **原则上已进行网上信息采集的不再现场审核资料**

### 2. 坚持免试入学
- 公办中小学：实行单校划片或多校划片方式入学
- 民办中小学：实行免试电脑随机录取
- **严禁事项**：
  - 严禁通过笔试、面试、面谈、评测、简历等任何形式选拔学生
  - 严禁以各类考试、竞赛、考测证书、荣誉证书、学科成绩、培训成绩或证书证明等作为招生入学依据或参考
  - 严禁中小学与培训机构、社会组织挂钩招生

### 3. 坚持阳光分班
- 实施均衡编班，确保男女生比例大体相当，师资配置合理均衡
- **不得以任何名义在校内分设重点班和非重点班**
- 严禁以学生考试成绩、获奖和竞赛等为依据编设重点班、特长班等
- 分班过程邀请教师代表、家长代表、责任督学参加，接受监督

### 4. 坚持公民同招
- 公办、民办义务教育阶段学校实行同步招生
- 统一纳入全市义务教育招生入学平台管理，实行网上报名和招生录取

### 民办学校入学政策优化
1. **支持优质民办学校增加招生计划**：2025年总报名人数超过总核准计划的学校、2025年直升报名人数已达到总核准计划的学校可适当增加招生计划
2. **扩大空余计划报名范围**：空余计划录取阶段，除符合在蓉入学条件且未被民办学校录取的学生外，符合合法稳定居住或合法稳定就业的随迁子女申请人也可报名参加空余计划录取

## 二、2026成都幼升小入学时间表

### （一）本市户籍适龄儿童小一入学流程

| 时间 | 事项 | 说明 |
|:---|:---|:---|
| **4月24日前** | 查看登记点 | 关注户籍地区(市)县教育行政部门的户籍入学登记公告 |
| **5月6日-12日** | 网上信息采集 | 登录"成都市义务教育招生入学服务平台"填报信息 |
| **5月13日-15日** | 现场资料审核（特殊情况） | 确需现场审核的到指定登记点进行资料审核 |
| **6月15日** | 入学结果查看 | 登录"入学平台"查看公办小学入学安排结果 |
| **7月5日前** | 学生报到 | 根据学校通知办理入学报到手续 |

> **特别提醒**：登记点是资料审核点，不等同于就读学校；划片范围由区(市)县于6月15日向社会公布

### （二）民办小学入学流程

| 时间 | 事项 | 说明 |
|:---|:---|:---|
| **5月6日-12日** | 取得报名资格（本市户籍） | 完成公办小学信息采集、校验通过 |
| **5月13日-15日** | 取得报名资格（现场审核） | 现场资料审核通过 |
| **4月1日-30日** | 取得报名资格（随迁子女） | 在"入学平台"申请 |
| **4月15日-5月9日** | 查询入学申请结果 | 随迁子女查询审核结果 |
| **5月31日前** | 取得报名资格（其他符合政策） | 在"入学平台"完成信息采集 |
| **6月1日-5日** | 网上报名 | 在"小学入学服务"中的"民办小学"报名模块选报1所民办小学 |
| **6月16日** | 查看录取结果 | 登录"入学平台"查看 |
| **6月17日-18日** | 学位确认 | 既有公办学位又有民办学位的需进行学位确认 |
| **7月5日前** | 学生报到 | 根据学校通知办理入学报到手续 |

> **重要提醒**：凡报民办小学均需要先进行信息采集，获得民办小学的报名资格。逾期未确认学位的，视为自动放弃民办学位，默认为公办学位。

## 三、2026成都小升初入学时间表

### （一）公办小升初入学流程

| 时间 | 事项 | 说明 |
|:---|:---|:---|
| **4月20日-24日** | 信息核对 | 本市小学毕业生在"入学平台"进行信息核对；随迁子女需于4月1日-30日申请；回蓉生进行信息采集；双胞胎(多胞胎)子女申请绑定随机录取 |
| **6月2日** | 查看升学对应区域 | 查看学生公办初中升学对应区域 |
| **6月15日-18日** | 市直属学校网上报名 | 如要报名市直属学校 |
| **6月23日** | 市直属学校录取结果 | 查看市直属学校电脑随机录取结果 |
| **7月2日** | 查看录取结果 | 查看区属公办初中录取结果 |
| **7月10日-11日** | 学生报到 | 按初中学校安排报到 |

### （二）民办初中入学流程

| 时间 | 事项 | 说明 |
|:---|:---|:---|
| **4月20日-24日** | 取得报名资格（本市毕业生） | 在"入学平台"进行信息核对 |
| **4月1日-30日** | 取得报名资格（随迁子女） | 在"入学平台"申请 |
| **4月15日-5月9日** | 查询入学申请结果 | 随迁子女查询审核结果 |
| **6月2日-3日** | 一贯制学校直升报名 | 已在民办一贯制学校就读小学拟直升初中的 |
| **6月5日** | 一贯制学校直升录取结果 | 查看录取结果 |
| **6月15日-18日** | 民办初中报名 | 在"入学平台"报名 |
| **6月24日** | 查看录取结果 | 查看区属民办初中录取结果 |
| **7月3日-4日** | 学位确认 | 被民办学校和区属公办学校同时录取的须进行网上学位确认 |
| **7月10日-11日** | 学生报到 | 按初中学校安排报到 |

> **温馨提示**：已自愿报名并被民办学校校内直升录取的学生，不再参加其他民办学校和公办学校录取。

> **中考政策提醒**：从2025年入学的初一新生起，就读民办初中的学生中考时若希望申请本校高中校内指标，须满足符合初中政策性入学、在实际就读学校有3年学籍(政策转学的须在学籍转入学校实际就读满2年)应届毕业生的条件。

## 二、办理步骤

### 步骤1：取得专业法律意见（8月底前）

- [ ] 联系互联网法律专业律师/律所
- [ ] 提供业务模式说明（K12升学政策研判工具+微信支付）
- [ ] 获取法律意见书，确认是否需要ICP许可证
- [ ] 如需要，确认申请类别（B25信息服务业务）

### 步骤2：准备申请材料

- [ ] 公司营业执照副本（需为四川注册企业）
- [ ] 公司章程
- [ ] 验资报告（注册资本不低于100万）
- [ ] 主要管理人员身份证及简历
- [ ] 办公场地证明（租赁合同或产权证）
- [ ] 域名证书（diandengwa.com）
- [ ] 服务器托管协议（腾讯云162.14.74.251）
- [ ] 信息安全管理制度
- [ ] 用户信息安全管理制度

### 步骤3：提交申请

- [ ] 向四川省通信管理局提交申请
- [ ] 等待审核（通常40-60个工作日）
- [ ] 取得许可证后，在网站页脚公示许可证编号

## 三、风险提示

- 未取得ICP许可证开展经营性互联网信息服务，可能面临罚款和关停
- 微信支付功能可能因无ICP许可证被限制
- 建议在许可证取得前，保持低风险运营（控制交易量）

## 五、替代方案

如短期内无法取得ICP许可证，可考虑：
1. 与已持有ICP许可证的平台合作（如入驻教育平台）
2. 将支付功能暂时下线，改为免费运营+邀请码模式
3. 将公司注册地变更到ICP许可证审批更快的地区
